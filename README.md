# custormer-churn-project
Predicting customer churn using machine learning models and business insights.

Project Overview :
    This project predicts customer churn in subscription-based businesses using classification models and identifies key factors influencing churn.

Problem Statement:
    -> Objective- Identify customers at high risk of churning and uncover key drivers influencing churn.
    -> Type- Binary Classification
    -> Target variable- churn(1= Yes, 0=No)

Business Goal:
    - Detect high-risk customers early
    - Help business teams design retention strategies
    - Reduce revenue loss
